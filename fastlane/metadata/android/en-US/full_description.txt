<p>This is an Android application that uses the <a href='https://pixabay.com/api/docs/' target='_blank' rel='nofollow noopener'>Pixabay API</a> to browse, search, and download images.</p><p><br><b>Features:</b></p><ul><li>🔍 Search images using keywords</li><li>🎨 Filter images by category and color</li><li>⬇️ Download images</li><li>🔄 Swipe to refresh content</li><li>🔍 Pinch-to-zoom on image previews</li><li>💾 Efficient image loading and caching</li><li>📱 Responsive UI</li></ul>
